# SEO & AI Search Optimization - Sauvage

## Optimizations Implemented

### 1. Structured Data (JSON-LD)
✅ **LocalBusiness Schema** - Helps Google, Bing, and AI models understand what Sauvage is
- Name, address, phone, email, coordinates
- Amenities & services listed
- Opening hours specified
- Aggregate ratings

✅ **Organization Schema** - Brand recognition across search engines
- Company name, logo, description
- Contact information

✅ **WebSite Schema** - AI search engine indexing
- Site name, description, search action

### 2. Meta Tags
✅ **Title & Description** - Already optimized for click-through rates
✅ **Open Graph Tags** - Social sharing preview (Facebook, LinkedIn, WhatsApp, etc.)
✅ **Twitter Card Tags** - Twitter/X preview
✅ **Robots Meta** - Tells search engines to fully index and show images/snippets

### 3. Semantic HTML
✅ **Proper Heading Hierarchy**
- H1: "Sauvage" (main page title)
- H2: Section titles (The Space, What's On, etc.)
- Proper nesting prevents confusion for AI/SEO

✅ **Semantic Elements**
- `<section>` tags for logical content areas
- `<article>` tags for independent content blocks
- `<nav>` for navigation
- Proper alt text on images

### 4. AI & Web Search Optimization
✅ **Crawler-Friendly Structure**
- Clear, semantic HTML
- No bloated CSS/JS overhead
- Fast load times (improves ranking)
- Mobile responsive

✅ **Content Optimization**
- Keywords naturally integrated: "event venue Amsterdam", "café Amsterdam West", "exhibition space", "natural wine bar"
- Long-tail keywords in descriptions
- Schema markup helps AI understand context

✅ **Mobile Optimization**
- Responsive design (mobile-first)
- Viewport meta tag configured
- Touch-friendly buttons and spacing

### 5. Technical SEO
✅ **Canonical Tag** - Prevents duplicate content issues
✅ **Language Tags** - hreflang for international SEO (currently en-US)
✅ **Favicon** - Brand recognition in search results and tabs
✅ **Theme Color** - Mobile browser UI matches brand
